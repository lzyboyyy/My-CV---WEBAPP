# Assignment 1 – Personal Portfolio Website

## Live concept
This project is a 9-page personal website built for Assignment 1.

## Site map
- index.html – Home
- about.html – About
- projects.html – Projects
- gallery.html – Gallery
- blog.html – Blog list
- post-1.html – Sample blog post
- contact.html – Full-featured form
- map.html – Map and video iframe
- policy.html – Policies and terms

## Features
- Semantic HTML5 structure
- Consistent navigation bar on all pages
- At least 5 images with alt text
- Figure and figcaption
- Google Maps iframe and YouTube iframe
- Full-featured form with all required input types
- External CSS file
- Responsive layout using Grid/Flexbox
- Accessibility support: skip link, focus states, good font size, heading order
- Small JavaScript for menu, range output, and friendly form validation

## Folder structure
- assets/css/styles.css
- assets/js/main.js
- assets/img/

## How to run
1. Open the project folder.
2. Double-click `index.html`, or run with VS Code Live Server.
3. To deploy online, upload the folder to GitHub Pages or Netlify.
